Template Converter

Define Template 1 and Template 2 using placeholders like {{field}}. Paste a filled message that matches Template 1 and convert it into Template 2.

Template 1 (input format)
Placeholders must be in double curly braces, for example {{name}}. All other text must match exactly.
Template 2 (output format)
Values captured from Template 1 will be inserted into Template 2 placeholders.
